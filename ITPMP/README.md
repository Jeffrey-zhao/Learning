#### 核心统计
   * 沟质整范进(狗子整范进)，成人风采 + 干系人 == 10大管理
   * 进成质范(进城吃饭)，风沟采人(疯狗踩人) +干系人 == 整体管理
   * 查看科目分值，总结出考核重点
